npm install
node build.js
npm start

For distributable builds:
npm run build:win
npm run build:mac
npm run build:linux
those use electron-builder to produce the portable .exe / .app / AppImage.

The .bat files are convenience wrappers:
  build-npm.bat       — rebundle renderer + launch (npm start)
  build-npm-fast.bat  — launch only, no rebundle (npx electron .)
  build-win.bat       — in-place Windows build into AMMViewerContents\ + AMMViewer.lnk
  build-win-zip.bat   — distributable dist\AMMViewer-win.zip
  build-mac.bat       — cross-build AMMViewer.app from Windows (tar.gz)
  build-linux.bat     — cross-build Linux unpacked dir (tar.gz)
